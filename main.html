Orion Framework
- Buttons -
Buttons Have Custom Attributes to Show.
These are type, color, hover and state.
BType Attribute
Defines the Button Shape:
fill (Default), border or ghost.
<div class="button text" btype="fill">Filled</div>

<div class="button text" btype="border">Border</div>

<div class="button text" btype="ghost">Ghost</div>;
Filled
Border
Ghost
Color Attribute
Changes the Button Color:
accent, success, danger, or warning.
Changes to Background With Custom Colors are Also Valid.
<div class="button text" btype="border">Default</div>

<div class="button text" color="accent">Accent</div>

<div class="button text" color="success" btype="border">Success</div>

<div class="button text" color="danger">Danger</div>

<div class="button text" color="warning" btype="border">Warning</div>

<div class="button text" style="background: #02bb9c;">Custom</div>
Default
Accent
Success
Danger
Warning
Custom
Hover Attribute
Changes the Hover Animation:
def (Default), float and sink.
You can add Multiple Hover Effects.
<div class="button text" hover="def">Default</div>

<div class="button text" hover="" btype="border">None</div>

<div class="button text" hover="float" btype="ghost">Floating</div>

<div class="button text" hover="def float">Default & Floating</div>

<div class="button text" hover="sink" btype="border">Sinking</div>

<div class="button text" hover="def sink" btype="ghost">Default & Sinking</div>
Default
None
Floating
Default & Floating
Sinking
Default & Sinking
Cursor Attribute
Changes the Mouse Cursor on Hover:
load or click.
<div class="button text">Default</div>

<div class="button text" cursor="load" btype="border">Loading</div>

<div class="button text" cursor="click" btype="ghost">Clickable</div>
Default
Loading
Clickable
