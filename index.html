Orion Framework v1.2.12
- Buttons -
Buttons Have Custom Attributes to Show.
These are shape, color, hover and state.
Shape Attribute
Defines the Button Shape:
plain (Default), shadow, border or ghost.
<div class="button text" shape="plain">Plain</div>

<div class="button text" shape="shadow">Shadow</div>

<div class="button text" shape="border">Border</div>

<div class="button text" shape="ghost">Ghost</div>;

Shadow

Color Attribute
Changes the Button Color:
accent, success, danger, or warning.
Changes to Background With the --oBackground Variable are Also Valid.
<div class="button text" shape="border">Default</div>

<div class="button text" color="accent">Accent</div>

<div class="button text" color="success" shape="border">Success</div>

<div class="button text" color="danger">Danger</div>

<div class="button text" color="warning" shape="border">Warning</div>

<div class="button text" style="--oBackground: #02bb9c;">Custom</div>

Hover Attribute
Changes the Hover Animation:
def (Default), float and sink.
You can add Multiple Hover Effects.
<div class="button text" hover="def">Default</div>

<div class="button text" hover="" shape="border">None</div>

<div class="button text" hover="float" shape="ghost">Floating</div>

<div class="button text" hover="def float">Default & Floating</div>

<div class="button text" hover="sink" shape="border">Sinking</div>

<div class="button text" hover="def sink" shape="ghost">Default & Sinking</div>

None

Cursor Attribute
Changes the Mouse Cursor on Hover:
click (Default), load or auto.
<div class="button text" cursor="click">Clickable</div>

<div class="button text" cursor="load" shape="border">Loading</div>

<div class="button text" cursor="auto" shape="ghost">Auto</div>
Clickable

Auto
