- BUTTON -
To create a button, use the o-button element. To modify the text, change it's innerHTML.
Button
<o-button></o-button>

<o-button style="width: 100px; height: 40px;"></o-button>

<o-button>Button</o-button>
COLORS
To personalize your button, use the background and color attributes to change the background and text color respectively.
Background Color
<o-button background="#4f3a50">Background</o-button>

<o-button color="#d4007c">Color</o-button>
SHAPE
You can also change your button's style with the shape attribute.
Shape value can set to plain (default), border and ghost.
Plain Border Ghost
<o-button shape="plain">Plain</o-button>

<o-button shape="border">Border</o-button>

<o-button shape="ghost">Ghost</o-button>
CONTENT
To prepare your button to it's content, you can use the content attribute.
Content value can set to text (default), textbox (text without padding), emote and box (makes it a container).
Additionally, to make the content vertical use the vertical attribute.
Text Textbox ü¶Ü box
<o-button content="text">Text</o-button>

<o-button content="textbox">Textbox</o-button>

<o-button content="emote">ü¶Ü</o-button>

<o-button content="box">box</o-button>
HOVER
To change your button's hover animation, you can use the hover attribute.
Hover value can set to def (default), float, sink and none.
Default Float Sink Default Float Default Sink None
<o-button hover="def">Default</o-button>

<o-button hover="float">Float</o-button>

<o-button hover="sink">Sink</o-button>

<o-button hover="def float">Default Float</o-button>

<o-button hover="def sink">Default Sink</o-button>

<o-button hover="none">None</o-button>
CURSOR
To change your button's cursor, you can use the cursor attribute.
Cursor value can set to click (default), load and auto.
Click Load Auto
<o-button cursor="click">Click</o-button>

<o-button cursor="load">Load</o-button>

<o-button cursor="auto">Auto</o-button>
ICONS
To add an icon to your button, use the lefticon attribute.
Left Icon
<o-button lefticon="Data/Images/logo.png">Left Icon</o-button>
- INPUT -
To create an input, use the o-input element. You can use the value attribute to change it's text.
<o-input></o-input>

<o-input value="Text"></o-input>
COLORS
To personalize your input, use the background and color attributes to change the background and text color respectively. Additionally, you can use the transparent attribute to make the surrounding box disappear.
<o-input background="#4f3a50"></o-input>

<o-input color="#d4007c" value="text"></o-input>

<o-input transparent value="text"></o-input>
TEXT
You can also use the placeholder and label attributes to change the placeholder and the label respectively. As the label only shows when there is no text, use the showlabel attribute to make it show permanently.
<o-input placeholder="Placeholder" label="Label"></o-input>

<o-input label="Label" showlabel></o-input>
TYPE
You can also change your input's type with the type attribute.
Type value can set to text (default), password, search, number, date, month, week and time.
<o-input type="text" label="Text" showlabel></o-input>

<o-input type="password" label="Password" showlabel></o-input>

<o-input type="number" label="Number" showlabel></o-input>

<o-input type="date" label="Date"></o-input>
OTHER
There are other attributes that can be applied to inputs.
These are max, which changes the maximum number of characters, and disabled, which disables the input.
<o-input max="10" label="Max" showlabel value="Max of 10"></o-input>

<o-input disabled label="Disabled" showlabel value="I am disabled üë©‚Äçü¶Ω"></o-input>

- EFFECTS -
Classes that can be added to elements to create effects.
GLOW
The glow class replaces the element's background to create a glow effect.
<div class="glow"></div>
NOISE
The noise class replaces the element's background to create a noise effect.
<div class="noise"></div>
GLASS
The glass class adds a shadow and blurs the background to create a glass effect.
<div class="glass"></div>
PULSE
The pulse class adds a shadow around an element to create a pulse effect.
Changing the --oColor variable applies a custom color.
<div class="pulse"></div>

<div class="pulse" style="--oColor: #d4007c;"></div>
JS
- NOTIFICATIONS -
To create a notification, use the createNoti(title, content, options) function.
Notifications will stack and keep appearing until there is none left.
Notification Test
TITLE & CONTENT
To personalize your notification, use the title and content (strings) to change the title and content respectively.
Notification Test
createNoti('Title', 'Content')
OPTIONS
To add extra personalization, the options (object) can add more things to your notification. Options value can contain:
  ‚Ä¢ An onClick (function) to run when clicked.
  ‚Ä¢ A time (number) to state the time in ms it will stay before disappearing.
Notification Test
createNoti('Test', 'Click me!', {
  onClick: function() { window.open('https://github.com/BOTPanzer') },
  time: 1500
})
- DIALOGS -
To create a dialog, use the createDialog(innerHTML, title, options) function, which will return the created dialog's ID.
Dialog Test
FUNCTIONS
These are some of the (functions) related to dialogs:
  ‚Ä¢ closeDialog(id) closes the dialog with the specified ID.
  ‚Ä¢ setDialogTitle(id, title) changes the title of the dialog with the specified ID.
let id = createDialog()
closeDialog(id)
setDialogTitle(id, 'Title')
TITLE & CONTENT
To personalize your dialog, use the innerHTML and title (strings) to change the HTML text you would write inside the dialog and the title respectively.
Dialog Test
createDialog(`<div style="width: 500px; padding: 20px; text-align: center;">
  This is a dialog example
</div>`, 'Title')
OPTIONS
To add extra personalization, the options (object) can add more things to your dialog.
Options value can contain:
  ‚Ä¢ An onClose (function) to replace the close function.
  ‚Ä¢ A preventClose (boolean) to prevent the user from closing the dialog.
Dialog Test
let id = ''
id = createDialog('innerHTML', 'Title', {
  onClose: function() {
    closeDialog(id)
    createNoti('Info', 'Dialog Closed')
  },
  preventClose: false
})

  innerHTML
