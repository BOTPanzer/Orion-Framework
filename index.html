- CHECKBOX & RADIO -
Elements That Mixed With the button Class Create Checkboxes.
• The --oColor Variable Applies a Custom Color.
CHECKBOX
The checkbox Class Applied to an input Element can be Used to Create Checkboxes.
• The checkbox Attribute Changes the Checkbox Type.
<input class="button checkbox" type="checkbox"/>

<input class="button checkbox" type="checkbox" radio="reverse"/>

<input class="button checkbox" type="checkbox" style="--oColor: #d4007c;"/>

<input class="button checkbox" type="checkbox" radio="reverse" style="--oColor: #d4007c;"/>
RADIO
The radio Class Applied to an input Element can be Used to Create Radio Buttons.
• The radio Attribute Changes the Radio Type.
<input class="button radio" type="radio"/>

<input class="button radio" type="radio" radio="reverse"/>

<input class="button radio" type="radio" style="--oColor: #d4007c;"/>

<input class="button radio" type="radio" radio="reverse" style="--oColor: #d4007c;"/>
- SWITCHES -
Elements That Mixed With the button Class Create Switches.
• The --oColor Variable Applies a Custom Color.
SWITCH
The switch Class Applied to an input Element can be Used to Create Switches.
• The switch Attribute Changes the Switch Type.
<input class="button switch" type="checkbox"/>

<input class="button switch" switch="small" type="checkbox"/>

<input class="button switch" type="checkbox" style="--oColor: #d4007c;"/>

SEEKBAR
The seekbar Class can be Used on Range Inputs to Create a Seekbar.
