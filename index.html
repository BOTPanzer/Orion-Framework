Oriøn Framework
Oriøn Framework is a simple framework intended to be used in Oriøn Assistant.
It was made with customization in mind.
Theme
You can change theme colors & settings by changing these CSS attributes.
Theme
--background: #18191c; The color of the background
--menu: var(--background); The color of the background in dialogs

Scrollbar
--scrollbar: #f5f5f569; The color of the scrollbar's background
--scrollbarThumb: #f5f5f5; The color of the scrollbar's thumb
--scrollbarSize: 16px; The size of the scrollbar

Colors
--accent: #cc1d4e; The accent color
--success: #00a023; A color to indicate success
--danger: #af0636; A color to indicate danger
--warning: #ce9700; A color to indicate warning
--progress: #C7C9CA; A color to indicate progress

Text
--text: #f5f5f5; The color of text
--textSecondary: #afb1b3; An alternate color to use in text
--textHint: var(--textSecondary); The color of text input hints
--textWindowButton: var(--text); The color of dialog close buttons text
--textSize: 15px; The size of the text

Buttons
--button: #202225; The color of buttons
--buttonBorderColor: transparent; The color of the border of buttons
--buttonBorderWidth: 0px; The width of the border of buttons
--buttonIconFilter: brightness(0) invert(); The filter of button icons

Window
--windowMarginTop: 0px; The margin used at the top in dialogs
--windowButtonHover: #2a2e33; The color of the window's minimize & maximize buttons
--windowExitHover: #ED4245; The color of the window's close button

Other
--hover: #ffffff10; The color of the hover effect
HTML
All of the elements from the framework that you can use.
Box
Boxes are the main element of the framework, every other element is an extension of them.
To create a box, use the o-box element.
Hover
You can disable the hover animation by using the hover attribute.
<o-box></o-box>

<o-box hover="none"></o-box>
Personalization
Boxes can be personalized by using the corner, width, height, background, color & transparent attributes.
Hi Hi
<o-box corner="10px"></o-box>

<o-box width="24px" height="24px"></o-box>

<o-box background="#4f3a50"></o-box>

<o-box color="accent">
  <span style="margin: auto;">Hi</span>
</o-box>

<o-box transparent>
  <span style="margin: auto;">Hi</span>
</o-box>
Buttons
To create a button, use the o-button element.
Text
You can add text to a button by using the text attribute.
<o-button></o-button>

<o-button text="This button has custom text"></o-button>
Type
You can change the style of buttons by using the type attribute.
<o-button text="This is the default button"></o-button>

<o-button type="border" text="This is a border button"></o-button>

<o-button type="ghost" text="This is a ghost button"></o-button>
Size
You can change the size of buttons by using the size attribute.
<o-button text="This is the default button"></o-button>

<o-button size="medium" text="This is a medium button"></o-button>

<o-button size="big" text="This is a big button"></o-button>
Icons
You can add icons to buttons by using the lefticon & righticon attributes.
<o-button text="This button has a left icon" lefticon="Docs/Images/ig.png"></o-button>

<o-button text="This button has a right icon" righticon="Docs/Images/git.png"></o-button>

<o-button text="This button has both icons" lefticon="Docs/Images/ig.png" righticon="Docs/Images/git.png"></o-button>
Personalization
Buttons can be personalized by using the background, color & nofilter attributes.
<o-button background="#4f3a50" text="Custom background color"></o-button>

<o-button color="accent" big text="Custom text color"></o-button>

<o-button nofilter lefticon="Docs/Images/ig.png" text="Icons without a color filter"></o-button>
Button arrays
You can join multiple buttons by using button arrays.
To create a button array, use the o-array element.
<o-array>
   <o-button text="First button"></o-button>

  <o-button text="Second button"></o-button>

  <o-button text="Third button"></o-button>
</o-array>
<o-array>
   <o-button lefticon="Docs/Images/pre.png"></o-button>

  <o-button lefticon="Docs/Images/pause.png"></o-button>

  <o-button lefticon="Docs/Images/next.png"></o-button>
</o-array>

Text inputs
To create a text input, use the o-input element.
Text
You can add text & hints to a text inputs by using the value & hint attributes.
<o-input></o-input>

<o-input value="Custom text"></o-input>

<o-input hint="Custom hint"></o-input>
Size
You can change the size of text inputs by using the size attribute.
<o-input value="Default"></o-input>

<o-input size="medium" value="Medium"></o-input>

<o-input size="big" value="Big"></o-input>
Label
You can add a label to text inputs by using the label attribute.
Adding the label attribute without a text will use the hint instead.
<o-input hint="Hint" label value="Text"></o-input>

<o-input hint="Hint" label="Medium" value="Text" size="medium"></o-input>

<o-input hint="Hint" label="Big" value="Text" size="big"></o-input>
Type
You can change the input type of text inputs by using the type attribute.
<o-input type="text" hint="Text"></o-input>

<o-input type="password" hint="Password"></o-input>

<o-input type="number" hint="Number"></o-input>
Max, readonly & disabled
You can change text inputs max amount of characters, make them read only and disabled by using the max, readonly & disabled attributes.
<o-input value="Max of 10" max="10"></o-input>

<o-input value="Read only" readonly></o-input>

<o-input value="Disabled 👩‍🦽" disabled></o-input>
Personalization
Text inputs can be personalized by using the background, color & transparent attributes.
<o-input value="Custom background" background="#4f3a50"></o-input>

<o-input value="Custom color" color="accent"></o-input>

<o-input value="Transparent" transparent></o-input>

JS
All of the functions from the framework that you can use.
Notifications
To create a notification, use the createNoti(title, content, options) function.
Title & content
You can modify the title & content of the notification by using the title & content string arguments.
createNoti('Title', 'Content')
Options
You can customize even further your notification by using the options object argument.
createNoti('Test', 'Click me!', {
  onClick: () => window.open('https://botpa.vercel.app/', '_blank'),
  duration: 3000
})
Context Menus
To create a context menu, use the createCTXMenu(title, items, title) function.
Returns the id of the context menu.
Event
In order to create a context menu, you will need the event variable from the event listener you are calling the function from.
const id = createCTXMenu(event)
Items & title
To add items and a title to your context menu, you can use the items array and the title string variables.
const id = createCTXMenu(event, [
  { id: 'test1', label: 'Label 1', click: () => {} },
  { id: 'test2', label: 'Label 2', click: () => {} }
], 'Title')
Functions
To close a context menu at will, you can use the closeCTXMenu(id) function.
const id = createCTXMenu(...)

closeCTXMenu(id)
Dialogs
To create a dialog, use the createDialog(content, title, options) function.
Returns the id of the dialog.
Content & title
To add content & a title to your dialog, you can use the content & title string variables.
The content is the plain text HTML inside of the dialog.
const content = '<div style="margin: 20px">This is the content of the dialog</div>'

const id = createDialog(content, 'Title')
Options
You can customize even further your dialog by using the options object argument.
There are options to replace the onClose function of the dialog and to hide the close button with hideClose.
const content = '<div style="margin: 20px">This is the content of the dialog</div>'

const id = createDialog(content, 'Title', {
  onClose: () => {
    closeDialog(id)
    createNoti('Info', 'Dialog closed')
  },
  hideClose: false
})
Functions
To rename or close a dialog at will, you can use the renameDialog(id, title) & closeDialog(id) functions.
const id = createDialog(...)

renameDialog(id, 'Title')

closeDialog(id)
Dialog Builder
You can use the DialogBuilder.build(type, options) function to make the dialog creation easier.
Returns an object with the contents of your dialog.
Info
You can use the DialogBuilder.INFO type to create an informational dialog.

Returns an object with:
· content: the dialog's plain text HTML
const type = DialogBuilder.INFO

const options = {
  content: 'Low damage, and the ability to jump over enemy walls...',
}

const db = DialogBuilder.build(type, options)

const id = createDialog(db.content)
Alert
You can use the DialogBuilder.ALERT type to create an alert dialog.

Returns an object with:
· content: the dialog's plain text HTML
· confirmId: the id of the confirm button
const type = DialogBuilder.ALERT

const options = {
  content: 'Low damage, and the ability to jump over enemy walls...',
}

const db = DialogBuilder.build(type, options)

const id = createDialog(db.content)

// Add listener to the confirm button using db.confirmId
Confirm
You can use the DialogBuilder.CONFIRM type to create a confirmation dialog.

Returns an object with:
· content: the dialog's plain text HTML
· confirmId: the id of the confirm button
· cancelId: the id of the cancel button
const type = DialogBuilder.CONFIRM

const options = {
  content: 'Low damage, and the ability to jump over enemy walls...',
}

const db = DialogBuilder.build(type, options)

const id = createDialog(db.content)

// Add listener to the confirm button using db.confirmId
// Add listener to the cancel button using db.cancelId
Input
You can use the DialogBuilder.INPUT type to create a text input dialog.

Returns an object with:
· content: the dialog's plain text HTML
· confirmId: the id of the confirm button
· inputId: the id of the text input
const type = DialogBuilder.INPUT

const options = {
  text: 'Text',
  hint: 'Hint',
}

const db = DialogBuilder.build(type, options)

const id = createDialog(db.content)

// Add listener to the confirm button using db.confirmId
// Do something with the text input using db.inputId
Custom
You can use the DialogBuilder.CUSTOM type to create a dialog with custom buttons.

Returns an object with:
· content: the dialog's plain text HTML
const type = DialogBuilder.CUSTOM

const options = {
  content: 'Low damage, and the ability to jump over enemy walls...',
  buttons: [
    {
      id: 'custom-save',
      text: 'Save'
    },
    {
      id: 'custom-cancel',
      text: 'Cancel'
    }
  ]
}

const db = DialogBuilder.build(type, options)

const id = createDialog(db.content)

// Add listeners to the custom buttons (in this case, custom-save & custom-cancel)
